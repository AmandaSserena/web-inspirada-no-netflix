Meu Portfólio

Sushi Parlour

Descrição: Site moderno e responsivo para Sushi Parlour, um restaurante japonês, com foco em representar a sofisticação da marca e facilitar o acesso ao cardápio, reservas e pedidos online.

Objetivos: Desenvolver um site visualmente atraente, intuitivo e otimizado para dispositivos móveis.

Tecnologias Usadas: HTML, CSS, JavaScript, Bootstrap (ou Tailwind CSS).

Soluções Implementadas: Layout responsivo, imagens de alta qualidade, animações interativas, integração de reservas e pedidos online.

Resultados: Aumento de 25% nas reservas online no primeiro mês após o lançamento.

Wallpaper Gallery
Imagens para Todos os Gostos

Descrição: Desenvolvimento de um site de Wallpaper Gallery, onde os usuários podem explorar e baixar uma vasta coleção de papéis de parede de alta qualidade, com categorias variadas para diferentes preferências.

Objetivos: Criar uma plataforma intuitiva e visualmente atraente para exibir imagens de papel de parede, permitindo fácil navegação e download.

Tecnologias Usadas: HTML, CSS, JavaScript, Bootstrap (ou Tailwind CSS), APIs para imagens (ex: Unsplash).

Soluções Implementadas: Design minimalista com categorias organizadas, sistema de busca para facilitar a localização de imagens, e integração com APIs para obter imagens em alta resolução.

Resultados: O site aumentou a interação dos usuários em 40%, com feedback positivo sobre a qualidade e organização das imagens.

Jornal Digital - Mundo Agora

Descrição: Criação de uma página de jornal digital, Mundo Agora, com foco em notícias atualizadas, design limpo e fácil navegação, ideal para leitores de todas as idades.

Objetivos: Desenvolver uma plataforma intuitiva para a leitura de notícias, com categorias organizadas e sistema de busca

Tecnologias Usadas: HTML, CSS, JavaScript, Bootstrap (ou Tailwind CSS).

Soluções Implementadas: Design responsivo, categorização de notícias, integração de um sistema de busca e animações suaves para transições de páginas.

Resultados: Feedback positivo dos usuários quanto à facilidade de navegação e leitura, com aumento de 30% no tempo médio de visita no site.
